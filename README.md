# Script para el monitoreo del Galileo

## Integrantes  
Melissa Millares  
Felipe Montoya  

## Proposito
Utilizamos bash para tomar de una board Intel (Galileo) los siguientes datos  
-Uso de CPU  
-Uso de RAM  
-Carga del sistema  
-Velocidad de descarga  
-Velocidad de carga  
-Almacenamiento disponible
